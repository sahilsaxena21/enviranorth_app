# Envira-North Knowledge Base — Index

This wiki is the authoritative reference for Envira-North product questions. Every answer must come from content in this folder. If the answer is not here, tell the user to contact Envira-North Customer Support (519-527-2198 / bigair@enviranorth.com) rather than guessing.

## How this wiki is organized

- **`products/`** — one `.md` per product SKU (fans)
- **`controls/`** — one `.md` per control package or VFD option

## Control packages (`controls/`)

- **Schneider VFD** — standalone NEMA4X fan-mounted drive; pairs with SmartAIR
- **SmartAIR** / **SmartAIR Multi-Fan** — touchscreen HMI, 1 or up to 10 fans, temperature + humidity automation
- **TouchAIR** — multi-fan touchscreen HMI, NEMA4X, up to 20 fans, BAS-ready
- **EssentialAIR** / **ZoneAIR** / **CommandAIR** / **ModAIR** — Lenze-VFD-based packages for wall-mount, LVC zone control, remote keypad, or ModBus integration respectively
- **LVC** — Low Voltage Control, 0-10V daisy-chain, up to 7 fans on one network
- **TFD-1** — temperature controller add-on for LVC
- **Wind Sensor** — automatic shutdown on high wind (mandatory in agricultural installs)
- **Jazz HMI** / **Jazz Multi-Fan HMI** — Jazz-fan-only controls (Jazz runs on its own network)

## Answering rules

1. Answer only from content in this wiki. Do not invent SKUs, specs, dimensions, part numbers, or prices.
2. If a question is about a product or feature that is not documented here, reply: *"I don't have that in my documentation — please contact Envira-North Customer Support at 519-527-2198 or bigair@enviranorth.com."*
3. Use plain English. Do not copy marketing phrases ("whisper quiet," "transform your building") verbatim — paraphrase to be concrete.
4. When citing specs, mention the SKU so the reader can verify.
5. When a question could apply to multiple SKUs (e.g., "how to install Sailfin"), point the user to the specific size's page and note that the installation steps are shared across sizes.
